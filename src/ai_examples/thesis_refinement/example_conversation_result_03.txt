\subsection{Phase I: Defining Project Success Criteria}
Before the project starts, the project deadline and provided resources are decided, in proportion to the defined expectations and requirements of the project. These expectations and requirements serve as the benchmark for the evaluation of the final outcomes. During the project success criteria evaluation phase, these criteria are used to identify requirements that necessitate refinement of the project to achieve the previously defined success criteria. The project is deemed unsuccessful if these criteria cannot be fulfilled with the allocated resources or within the specified deadline.
